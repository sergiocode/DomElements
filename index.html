M06_06_FT_01 - Elements del DOM

Ésta web es una demostración del Document Object Model (DOM), ya que para hacerla ha sido necesario modificar el estilo de los elementos del DOM y a la vez añadir eventos a estos. Podemos dividirla en 4 apartados:

  • DOM Style
  • DOM Element
  • DOM Document
  • DOM Events


DOM Style

DIV

Este DIV contiene las siguientes propiedades de estilo:

  • width: 5em
  • height: 5em
  • background-color: red
  • border-radius: 20%
  • margin: 0em 3em

Propiedades como éstas nos permiten moldear cualquier elemento del DOM. En éste caso le he aplicado dimensiones de altura y anchura, color de fondo, margen y borde redondeado.



También podemos aplicar estilo a elementos del DOM que contengan texto:

Texto de ejemplo
 
#txtEx1 {
    color: pink;
    font-size: medium;
}
Texto de ejemplo
 
#txtEx2 {
    color: aquamarine;
    font-weight: bolder;
    font-family: 'Courier New', Courier, monospace;
}
Texto de ejemplo
 
#txtEx3 {
    color: coral;
    font-weight: bolder;
    font-family: 'Times New Roman', Times, serif;
    text-decoration: line-through;
}

DOM Element

Con Javascript podemos encontrar elementos del DOM gracias a los ID, tags, clases, selectores de CSS y colecciones de objetos HTML.

Ejemplo
ID 
document.getElementById(elemento);
Tags 
document.getElementByTagName(tag);
Clases 
document.getElementByClassName(nombre);
Selector CSS 
document.querySelector(selector);


DOM Document

Podemos añadir, modificar y eliminar contenido del DOM gracias a funciones de Javascript

Crear elemento HTML 
document.createElement(element);
Eliminar elemento HTML 
document.removeChild(element);
Añadir elemento HTML 
document.appendChild(element);
Reemplazar elemento HTML 
document.replaceChild(nuevo, antiguo);
Modificar contenido HTML 
element.innerHTML = contenido;
Modificar estilo 
document.style.propiedad = valor;


DOM Events y Demostración

Los eventos nos permiten crear web dinámicas y navegar por los nodos del DOM. Existen eventos de diferentes tipos, como por ejemplo de raton: click, mouseover, mousedown, mouseenter... También hay eventos para teclas e incluso para estados (como el evento load).

Para realizar estas demostraciones he tenido que hacer uso de los DOM Events, que han sido añadidos a un DOM Element gracias a la función addEventListener(), asignándoles un evento de click.

Generadores de elementos

A continuación encontrarás un botón que permite generar nuevos elementos HTML. Éste en concreto está programado para generar divs con una clase llamada "ejNewElement" con un estilo predefinido. Pruébalo!

Código utilizado:

btnAddElement.addEventListener('click', () => {
    var div = document.createElement('div');
    div.classList.add('ejNewElement');
    div.innerHTML = 'Nuevo div';
    div.setAttribute("id", autoDivId);
    autoDivId += 1;
    div.setAttribute('onClick', "identificarElemento()");
    addDiv.appendChild(div);
});
